# FoodHub 
🍔Simple food delivery web app jisme multiple restaurants, cart management aur order tracking ka system hai. Isko vanilla JavaScript, HTML aur CSS se banaya hai.🚀
# Naye Features (Updates)
* Email & Mobile Login/Signup: Ab users apna account mobile number ya email dono se bana sakte hain aur login bhi kar sakte hain. 
* Distance-Based Delivery Fee: Delivery charge ab location ke kilometer ke hisab se calculate hota hai (jaise base fee + extra per km charge). 
* Profile Management: User apni profile details (naam, email, mobile) update kar sakte hain. 
# 🛠️ Tech Stack
* HTML5, CSS3
* JavaScript (ES6)
* LocalStorage (data save karne ke liye)
# 📂 Files
* index.html - Main HTML structure
* style.css - Styling aur responsive design
* app.js - Saara logic, routing aur cart functions
# 💻 Kaise Chalayein?
* Repo download ya clone karo.
* index.html ko browser mein open karo (ya VS Code ke Live Server se chalao).
